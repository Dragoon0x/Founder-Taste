# CTA Design

The CTA is the most important element on your page. Treat it that way.

## How to use

- `/cta-design` Evaluate and fix CTAs in any page or screen in this conversation.

## Constraints

### One Primary CTA Per Screen
- MUST have exactly one primary CTA per viewport. Not two. Not three. One.
- If you need a secondary action, make it visually quieter. Primary = solid button with brand color. Secondary = outlined or text-only link.
- MUST make the CTA the most visually prominent element in its section. Bigger, bolder, more contrast than anything around it.
- NEVER place two equally-styled buttons side by side. One must clearly be primary.

### Button Text
- MUST use action verbs that describe what happens next. "Start Free Trial," "Create Account," "See Pricing."
- MUST be specific. "Get Started" is acceptable. "Start Your Free Dashboard" is better.
- SHOULD reduce anxiety in the text. "Try Free — No Credit Card" removes a barrier. "Start Free Trial" alone leaves questions.
- NEVER use "Submit." Nobody wants to submit anything. Name the actual outcome.
- NEVER use "Click Here." It describes the mechanic, not the value.

### Visual Design
- MUST use your primary brand color for the CTA background. White or contrasting text on top.
- MUST ensure minimum 44px height for touch targets, 48px is better.
- MUST add generous horizontal padding (24-32px). Skinny buttons look cheap.
- SHOULD add 16-32px of whitespace around the button. Breathing room makes it more prominent.
- SHOULD use subtle hover effects: slight darkening or shadow. Confirms it's clickable.

### Placement
- Primary CTA in the hero: always.
- Repeat the CTA after major content sections. If someone reads your features and is convinced, don't make them scroll back up.
- Sticky CTA on mobile for critical conversion pages: a fixed bottom bar with the primary action.

### Anti-Patterns
- Ghost/outlined buttons as the primary CTA (they don't look clickable enough)
- CTA that blends into the page because the color matches the background
- "Learn More" as a primary CTA (it's a secondary action at best)
- Multiple CTAs with no visual difference between primary and secondary
- CTA below the fold on mobile with no way to reach it without scrolling
